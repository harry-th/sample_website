Important HTML tags

This page is desgined as a quick referencefor some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn hot to make HTML tables via table and related tags.

The tables below don't include all the HTML tags, but they do list many of the most important ones.

Block Elements

tag name purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
header header add a header
